About Dexoron CDN

Welcome to Dexoron CDN — a fast and reliable content delivery network offering free fonts, CSS styles, and icon libraries. Created by the community and enriched with my personal contributions, this project aims to simplify web development with high-quality resources.

Currently in development, Dexoron CDN already provides usable fonts and styles. Stay tuned for updates as we expand our collection!

How to Use

Integrate our resources into your project with a single line of code. Below are some examples:

Montserrat Font

A versatile, modern sans-serif font for any design.

<link rel="stylesheet" href="https://cdn.probrovova.com/fonts/Montserrat/main.css">

FontAwesome 6 Pro Icons

Premium icons for enhancing your UI.

<link rel="stylesheet" href="https://cdn.probrovova.com/libs/fontawesome/6/js/all.min.css">

DexoronCSS

Custom CSS framework for rapid styling (in progress).

<link rel="stylesheet" href="https://cdn.probrovova.com/css/DexoronCSS/all.css">
